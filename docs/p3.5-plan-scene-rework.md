## Phase 3.6-A / B / C — RESOLVED this session (2026-07-04)

All three are user-confirmed. Recorded here as the load-bearing architecture the rest of the phase must respect.

### 3.6-A — RD coil ridge system (ridges are a baked texture now)
The warped-sine flow field is deleted. `reaction-diffusion.ts` warms a 512² Gray-Scott **maze** (`feed 0.029 / kill 0.057`, dense-speckle seed) and bakes it once (`coil-bake.glsl`) into an RGBA8 texture: **R** crisp height, **G** gaussian-blurred (the C1 displacement channel), **B** mottle, **A** strands. `shell-shape.glsl` samples it via `coilUv` (full-turn equirect, seam in the X=0 groove). `coilHeight`/`coilHeightSmooth` replace `ridgeProfile`/`ridgeProfileSmooth`; the vertex stage's **Nyquist phase-step block is gone** (the bake is band-limited in texel space). Rule going forward: **every fragment-stage `texture2D` stays in uniform control flow** — an implicit-lod tap inside a divergent branch is ES3 undefined behavior (this was the "brown ring"). Vertex taps may branch (lod 0).

### 3.6-B — Macro-sculpt harness (the side-view rework, done via sliders)
The whole silhouette is uniforms now (`shell-params.ts`, 19 fields, `applyShellParams`), driven live on BOTH the preview AND the real site by a leva panel (`shell-dev-tools.tsx` + `shell-live-params.ts` override + a per-frame apply in `SurfaceScene`). What shipped in `crown` (= `SHELL_DEFAULTS`, user-blessed): superellipsoid `uBoxiness` slab halves, top-heavy `radialProfile` (shoulder bulge + base tuck), `uProfileFlip=1` (flat plateau up), `grooveRearFade=1` (halves merge over the rear), narrow **knife crease** (`cleftDepth 1.5`, `cleftWidth 0.02`, `moundHeight 0`) confined to the upper region (`grooveUnderFade = smoothstep(-0.25, 0.2, dir.y)` — the fix for "diverging halves / bottom-of-form read"), sub-mass with its own banded register, and the stalk (see 3.6-C). Presets `loaf`/`crown`/`bluff`. **Conventions preserved** (camera depends on them): long axis Z, groove in X=0, crown +Y.

### 3.6-C — Interior artifacts + the stalk architecture (the key structural call)
Interior is user-confirmed "completely fixed." Four stacked causes, unmasked in order: brown ring (ES3 UB, above) → melt oval (banded-register wavelength under the FD step → coarsened to ~0.46 u) → starburst (fiber-register azimuth degeneracy → `fiberBlendK` confines fibers to the column walls) → the structural root:

> **A stalk MUST NOT be a displacement of the main shell.** A single surface cannot protrude a column outward without digging a matching pit into the interior wall. The main shell runs `uStalkHeight = 0` (interior stays a clean continuous sheet); the stalk is a **companion pass** — `stalk-mesh.tsx`, a second copy of the same sphere/shader/params with `uStalkSolo = 1`, whose vertex stage collapses everything outside the stalk footprint and whose fragment stage discards it, so only the continuous column renders. `applyShellParams(m, p, stalkOn)` gates the stalk term; only the companion passes `true`.

This principle generalizes: **do not add outward-protruding features to the shared shell height field.** New masses that must not touch the interior get the companion-pass treatment.

## Verified load-bearing facts

- The cleft is keyed purely on `|dir.x|` (`shell-shape.glsl:46,54`) — a great circle in the X=0 plane. Re-orienting the anisotropic scale (long axis → Z, flatten Y) makes the *same* formula read as front-to-back over the top. The resculpt is a rescale + notch/lobe removal, **not** a geometry rewrite.
- The meridian stripe axis is baked into `atan(dir.z, |dir.x|)` evaluated in **three places** that must stay consistent: `ridgeProfile` (`shell-shape.glsl:113`), `detailHeight` (`:136`), and the frag AA phase (`tissue-shell.frag.glsl:48`).
- The dissolve aperture is hard-coded to world +z on **two** axes: `front = smoothstep(-0.35, 0.7, normalize(vWorldPos).z)` AND `radial = length(vWorldPos.xy)` (`tissue-shell.frag.glsl:152-153`). Both must follow the authored approach axis.
- RD texture is sampled as a flat planar projection `vWorldPos.xy*0.04+0.5` (`:142`) — smears the moment the camera sees the flanks. Needs a dir-based equirect UV.
- The shell's self-fog uniforms (`uFogColor/uFogDensity`) are set once and never updated — invisible today (density constant), visibly wrong once fog density becomes depth-driven. Fix required, not optional.
- Per-section scroll runway is already freely variable: `rawProgressToDepth` remaps arbitrary DOM heights onto fixed canonical bands, with ResizeObserver + fonts.ready re-measure. Only `ScaleSection`'s `full` (pinned 100vh) needs changing.
- No intro infrastructure exists anywhere (no component, no store flag, no scroll lock). `DepthIndicator` has live keyboard/click scale-jump nav that must be guarded during the overture.
- `content/sections/brain.md` has an unused `subtitle` frontmatter field — the arrival block's one-liner, already authored.
- `frameloop="demand"`: any new animation (intro timeline, parallax spring, mote drift) must call `invalidate()` or ride the ambient ticker.
