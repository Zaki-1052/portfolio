## What This Repo Does (One Paragraph)

This repo analyzes how SARS-CoV-2's main protease (MPro) moves and communicates internally under different drug conditions, using data from molecular dynamics simulations. The code parses allosteric pathway data (WISP), measures structural dynamics (RMSD/RMSF/PCA), and generates 3D visualizations (ChimeraX). The graduate student Javier works primarily out of `utils/`, which is the shared library that all notebooks import from.

## Glossary

### Biology Terms

| Term | Definition |
|------|-----------|
| **Active site** | The pocket on MPro where catalysis (substrate cutting) happens. Sits between Domain I and Domain II. |
| **Allosteric** | A change at one site on a protein affecting a distant site. The central research question here. |
| **Amino acid** | One of 20 building blocks of proteins. Each has a one-letter code (e.g., C = cysteine, H = histidine). |
| **Apo** | Unbound state -- no ligand. The control condition. |
| **CA (C-alpha)** | The central carbon of each amino acid's backbone. Used as the representative atom for a residue in most analyses. |
| **Catalytic dyad** | C145 and H41 -- the two residues that do MPro's cutting. Their distance indicates catalytic readiness. |
| **Chain A / Chain B** | In the dimer, the two copies of MPro. Chain A = residues 1-306, Chain B = 307-612. |
| **Conserved residue** | A position identical across all coronavirus main proteases (86 positions in MPro). |
| **Dimer** | Two copies of MPro paired together (its functional form). 612 residues total. |
| **Dimer_asym** | Asymmetric dimer -- drug bound to only one chain. No apo dimer_asym exists. |
| **Domain** | A structurally distinct region: Domain I (17-109), Domain II (110-175), IDL (176-200), Domain III (201-289). |
| **Ensitrelvir (ens)** | Non-covalent MPro inhibitor (Xocova). Sits in the pocket without bonding to C145. |
| **Force field** | The physics equations describing atomic interactions (e.g., ff14SB for protein, GAFF2 for ligands). |
| **IDL** | Interdomain linker (residues 176-200). Flexible loop connecting Domain II to Domain III. |
| **Ligand** | Any molecule that binds to a protein. Here: nirmatrelvir, ensitrelvir, or native peptide. |
| **Monomer** | A single copy of MPro (306 residues). |
| **MPro** | Main Protease of SARS-CoV-2. Also called 3CLpro or Nsp5. The drug target. |
| **Native peptide (nat)** | The natural substrate MPro cuts. Represents the "working" state. |
| **Nirmatrelvir (nirm)** | Covalent MPro inhibitor (active ingredient of Paxlovid). Bonds to C145. |
| **Protease** | An enzyme that cuts other proteins. MPro cuts the SARS-CoV-2 polyprotein. |
| **Residue** | An amino acid in context of a protein chain, numbered by position (e.g., residue 145 = C145). |
| **Subpockets** | Named pockets in the active site: S1 (140,141,163,172), S1' (24-26,142-145), S2 (41,49,54,189), S3/S4 (165-168,190-192). |
| **Topology** | File describing atom types and bonds (`.prmtop` for Amber). Paired with trajectories. |
| **Trajectory** | Recording of atom positions over time (`.nc` for Amber NetCDF). The raw simulation data. |

### Analysis Tool Terms

| Term | What It Does |
|------|-------------|
| **CNA** | Community Network Analysis -- partitions residue correlation network into communities. |
| **cpptraj** | AmberTools command-line tool for trajectory processing (distance matrices, correlation matrices). |
| **ChimeraX** | 3D molecular visualization program. Reads `.cxc` command scripts generated by this repo. |
| **MDAnalysis** | Python library for reading and analyzing MD trajectories. Core dependency. |
| **MM-GBSA** | Molecular Mechanics Generalized Born Surface Area -- estimates binding free energy. |
| **NANI** | N-Ary Natural Initialization -- a k-means clustering variant from the `mdance` library. |
| **PCA** | Principal Component Analysis -- finds dominant motion modes from trajectory data. |
| **POVME** | Pocket Volume Measurer -- computes binding pocket volume from structure snapshots. |
| **RMSD** | Root Mean Square Deviation -- measures overall structural drift from starting point. |
| **RMSF** | Root Mean Square Fluctuation -- measures per-residue flexibility over time. |
| **SEEKR** | Simulation Enabled Estimation of Kinetic Rates -- computes binding/unbinding rates. |
| **VMD** | Visual Molecular Dynamics -- another 3D visualization program. Reads `.tcl` scripts. |
| **WISP** | Weighted Implementation of Suboptimal Paths -- finds allosteric communication pathways. |

## Core Pipeline

### Flow 1: WISP Pathway Analysis

```
MD trajectories (.nc)
    │
    ▼  cpptraj (wisp/run_ptraj_test2.py)
distance matrix + correlation matrix
    │
    ▼  WISP (wisp/run_wisp_all.py or wisp_cct.py)
edge_usage.txt, node_usage.txt, hub_nodes.txt, critical_edges.txt
    │
    ▼  preprocessing.Parser.parse_txt() + build_df()
Enriched DataFrames (ligand, state, chain, domain, weight, ...)
    │
    ▼  processing.Filter / process_df()
Filtered DataFrames (top 10%, common/unique edges)
    │
    ▼  visualize.MProVisualizer.show_cylinder()
.cxc ChimeraX command scripts
```

### Flow 2: MD Trajectory Analysis

```
.prmtop + .nc files
    │
    ▼  mda.make_universe()
MDAnalysis Universe objects (ref + 3 replicas)
    │
    ├──▶ get_per_rep_rmsd()  → RMSD time series
    ├──▶ calc_rmsf()         → per-residue flexibility
    ├──▶ calc_dist()         → specific atom pair distances
    └──▶ PCA / POVME / etc.  → specialized analyses
    │
    ▼  matplotlib
Publication figures (.png at 300 DPI)
```

## Key Patterns

**System naming:** `mpro_{ligand}_{state}` -- e.g., `mpro_nirm_dimer`, `mpro_ens_monomer`, `mpro_apo_dimer_asym`

**Ligand/state iteration:**
```python
ligands = ['apo', 'nirm', 'ens', 'nat']
states = ['monomer', 'dimer', 'dimer_asym']
# Skip: apo + dimer_asym (no such system)
```

**Chain A/B offset:** Chain A = residues 1-306, Chain B = 307-612. Add 306 for Chain B. Code like `if res > 306: res -= 306` converts Chain B to Chain A numbering ("flattening").

**Interchain edges:** An edge with one residue in A and one in B has `chain='I'`.

**Weight normalization:** `normalized_weight` = [0, 1] via min-max. `weight_chx` = [0.2, 1.2] for ChimeraX cylinder radii.

**Import styles (inconsistent -- needs cleanup):**
- `from selections import ...` (works when `utils/` is on sys.path)
- `from utils.selections import ...` (works when parent of `utils/` is on sys.path)

**Environment variables:** `DATA_PROJECT_PATH`, `WISP_DATA_PATH`, `DATA_MD_PATH`, etc. with fallbacks to relative paths. Trajectory data lives on HPC, not in this repo.

## Domain Quick Reference

|--------|----------|-------------|
| N-term | 1-16 | Dimer interface, conserved |
| Domain I | 17-109 | Beta-barrel, contains lid loop (39-55) |
| Domain II | 110-175 | Beta-barrel, catalytic residues, substrate pockets |
| IDL | 176-200 | Flexible linker |
| Domain III | 201-289 | Alpha-helical, dimerization |
| C-term | 290-306 | Dimer interface |

**Catalytic dyad:** C145 (nucleophile) + H41 (general base)
