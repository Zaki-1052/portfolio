Lowercase monospace pill for domain and tech labels inside project cards and prose runs.

```jsx
<Tag>R</Tag> <Tag>mariner</Tag> <Tag tone="accent">epigenetics</Tag>
```

`tone="muted"` (default, hairline) or `tone="accent"` (tinted to the scale). Sharp corners regardless of scale. Keep labels short and lowercase; group them in a flex row with `gap`.
