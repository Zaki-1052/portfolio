## What was done

**Phase 3.5 — camera language + shell resculpt (complete)**
- **Camera keyframe schema rework** (`camera-keyframes.ts`): replaced per-knot `quaternion` with
  `position + target + roll` authoring; new pure `lookAtQuaternion()` (Shepperd basis→quat, with a
  parallel-up guard for the elevated establish shot) derives orientation each sample. Position AND
  target both Hermite-splined; `slerp` deleted. `REDUCED_ANCHOR_KEYFRAMES` (5 anchors) is the
  reduced-motion snap track. Dev panel gained target/roll(deg) fields. 14 keyframe tests (schema,
  lookAt guard, C1 continuity on every interior knot/axis, decaying-magnification invariant).
- **Shell resculpt** (`shell-shape.glsl`, `tissue-shell.vert.glsl`): `SHAPE` → oblong loaf
  (long axis Z 1.32, flat Y 0.78); `TAPER`/`BOTTOM_FLAT` give a wide level base; removed the old
  notch + LOBE_L/R foot bumps; central groove now reads front-to-back over the crown; per-half
  asymmetry breaks the mirror. Ridge field rewritten to a wandering **flow field** (two linear
  coords u = groove-distance, v = tilted latitude, vector domain-warp, interlock fold) so coils
  turn/loop/pack instead of vertical fluting. RD promoted to a `dir`-based equirect UV.

**Phase 3.6 — restructure, timing, resolution (mostly complete; ONE piece outstanding)**
- **Seventh "approach" band** (user request): new pure-3D journey band `[0, 0.14]` before the six
  content scales. `SCALES`/`SCALE_BOUNDARIES` grew to 7; `CONTENT_SCALES` added; new
  `ApproachContent.tsx` (tall empty scroll runway, `#approach` height in CSS, WebGL-gated to 230vh);
  `SurfaceScene` mounts across BOTH `approach` + `tissue` (scene-manager dedupes by component with a
  stable key so crossing the band boundary never remounts / re-warms RD). Depth indicator excludes
  approach (six-dot map, hidden during the journey). All boundary/store/registry tests re-pinned.
- **Breakthrough window is now derived** from `SCALE_BOUNDARIES[1]` and re-timed to fire LATE in the
  tissue band (~0.25→0.65 of the band): the whole hero plays with the form still OUTSIDE, hovering
  vertically over the groove; the plunge fires as the hero scrolls off and lands inside. Iterated the
  camera choreography with the user ~6× to lock the feel (pan completes in approach → overhead vertical
  slot at hero center → straight zoom-in, no lateral drift). `PLUNGE_APERTURE_DIR` derived from the
  camera position at `BREAKTHROUGH_START`; frag dissolve `front`/`radial` now follow `uApertureDir`
  (was hardcoded +z). **Fog / post-processing are NOT visibly working** — the user reports "no fog or
  effects." `fog-density.ts` + `scene-fog.ts` were added to make the shell's exp2 track the scene, but
  the dead-uniform bug is NOT confirmed fixed and the atmosphere reads as absent. UNRESOLVED — see open items.
- **Resolution rework (the important structural fix):** the displacement was tearing into triangle
  shards at grazing angles. Root cause: the SHARP shading profile (`pow`, unbounded slope) was being
  used for VERTEX displacement. Fix = two profiles from one stripe field — `ridgeProfileSmooth`
  (C1 smoothstep bump, bounded slope) drives geometry; `ridgeProfile` (crisp) stays fragment-only —
  PLUS a **Nyquist frequency-clamp** in the vertex stage: measure the flow field's actual phase-step
  across one mesh edge and fade relief out where the warp compresses stripes past the mesh limit.
  Triangle shredding is fully fixed. **The interior floor blurry-disc is NOT fixed** — a `floorness`
  mask + planar-RD sample + 2-octave backface wrinkle + grazing-damp were attempted in
  `tissue-shell.frag.glsl`, but the user confirms the smeared circle on the inner floor (from
  `inside.png`) is "still as bad as ever." The attempt did not work; treat as UNRESOLVED. See open items.

## Decisions
- Approach band gets no depth-indicator dot (it's the overture's extension; SPEC §7 says the indicator
  appears on arrival). Six-dot map = the content descent only.
- `SurfaceScene` deliberately spans two bands (one component, stable key) rather than a separate
  interior stub — the same shell IS the interior after the plunge; a stub would double-mount + re-warm RD.
- Geometry stays icosahedron detail 64 (~84.5k tris, 1 draw call): the Nyquist clamp makes raw
  subdivision the wrong lever — the flow field will always out-compress any finite mesh in pockets, so
  the vertex stage yields there and the fwidth-AA'd fragment normal carries the detail.
