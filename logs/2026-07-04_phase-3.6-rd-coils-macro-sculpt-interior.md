# 2026-07-04 — Phase 3.6 finish: RD coil ridges, macro-sculpt harness, interior artifacts resolved

> Neutral-language convention holds: "the shell" / "the form" / "coils" / "groove" / "sub-mass" / "stalk". No life-science vocabulary anywhere.

## What was done

**RD coil ridge system (3.6-A) — replaced the sin-flow field entirely.**
The ridges are now the Gray-Scott sim itself, promoted from faint mottle to the height source. Warmed in the **maze regime** (`feed 0.029 / kill 0.057` — the old coral params only ever grew rings), dense-speckle seeded (`1/150` texels) so the labyrinth matures everywhere at once (~0.6 s), then **baked once** (`coil-bake.glsl`) into an RGBA8 texture: **R** = crisp coil height (shading + AO), **G** = gaussian-blurred (C1 displacement), **B** = mottle, **A** = fine strands (iso-contours of the chem field — run along each tube for free). The sim is disposed after bake. The **entire Nyquist-clamp apparatus is deleted** — the bake band-limits displacement in texel space, so the mesh can't tear by construction. Full-turn equirect UV (`atan(dir.x,dir.z)`, 2 aspect-balanced repeats) with the seam parked in the X=0 groove strip. Sampler: `useCoilTexture`.

**Macro-sculpt harness (3.6-B) — the whole silhouette is now parameterized + live-tunable.**
`shell-params.ts` (19 params + `applyShellParams` mapper) drives a leva panel (`shell-dev-tools.tsx`) that works in BOTH `tissue-preview` AND on the live site (via `shell-live-params.ts` override channel + a per-frame apply in `SurfaceScene`). New shape controls in `shell-shape.glsl`: superellipsoid `uBoxiness` (slab halves), top-heavy `radialProfile` (shoulder bulge + base tuck), `uProfileFlip` (flat face down↔flat plateau up — the "flip it upside down" instinct), `grooveRearFade` (groove closes over the rear so the halves read merged), `uOverhang`, sub-mass + separation fold + its own banded texture register, `uFrontLift`. Presets: `loaf` (pre-rework baseline) / `crown` (shipped default) / `bluff`.

**The "peanut / diverging halves" fix.** The split's vertical envelope was full-height → two towers touching. `grooveUnderFade` now confines it to the upper region (`smoothstep(-0.25, 0.2, dir.y)`); `cleftDepth`/`moundHeight` dropped to a knife crease. The form reads as ONE joined mass a line divides toward the crown.

**Interior artifacts — resolved, user-confirmed "completely fixed".** A four-layer onion, each unmasking the next:
1. **Brown ring** — `texture2D` (implicit-lod) inside divergent `if` branches is ES3 undefined behavior; the user's GL backend resolved it as deepest-mip flat brown. Playwright's backend hid it (→ my earlier "fixes" were hallucinated). All fragment taps are now **unconditional**; blend factors gate.
2. **Melt oval** — banded-register wavelength (~0.29 u) fell under the FD normal step → coarsened to ~0.46 u.
3. **Starburst** — the fiber register's azimuth degenerates at the stalk axis → `fiberBlendK` confines fibers to the column walls.
4. **Structural root** — any stalk displaced into the shared sheet digs a matching interior pit. **Final architecture:** the main shell runs `uStalkHeight = 0` (clean continuous interior); the stalk is a **companion pass** of the same field (`stalk-mesh.tsx`, `uStalkSolo` collapses everything outside the footprint + discards) — the beloved continuous-column look, interior untouched by construction.

**Detail restored.** Crest strands (texture alpha) + groove-wall pleats (`pleats`/`grooveWall`) so the crack no longer reads unrendered at close framing. Stalk carries a cylindrical fiber register (`coilUvStalk`).

**Camera pull-back.** Hero/plunge altitude raised twice (user: too close, then "half that") and targets moved to look straight down the crease. Timing invariants still pass; the user re-confirmed "PERFECT."

## Decisions
- **Stalk = companion mesh pass, NOT a shell displacement** — a single surface can't protrude outward without an interior pit. This is the load-bearing architectural call of the session.
- All fragment-stage texture taps stay in **uniform control flow** forever (ES3 UB rule). Vertex taps may branch (lod 0, no derivatives).
- The macro form is a live instrument (sliders on the real site), not hand-tuned constants — the user iterates and blesses; `SHELL_DEFAULTS = crown` holds the blessed values.
